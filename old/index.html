다양한 기능

기능 #1

테러 방지 기능, 많이 들어보셨을거에요. 하지만 실질적으로 서버 테러를 막아주는 봇은 흔치 않죠. 이 봇은 서버 테러를 자동으로 보호하며 이상한 활동이 감지되면 자동으로 유저를 격리합니다!

기능 #2

서버 멤버 분들의 경고를 관리하느라 힘든 적이 있으실거에요. 이 봇은 서버 유저들에게 경고를 자동으로 부여하는 기능과 수동으로 부여하는 기능 모두 구현되어 있어 더욱 편리합니다!

기능 #3

음악 봇, AI 채팅 봇 등을 일일이 추가해야 해서 귀찮은 적이 한두번이 아닐 거에요. 이 봇은 그러한 기능들을 모두 통합하고 있습니다!

기능 #4

이 외에도 많은 기능이 있으니 서버에 봇을 추가하여 !help 명령어를 입력해 보세요!
